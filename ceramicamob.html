Ceramica Exhibition System

Professional Management for Large Scale Ceramic Events

Administrative Control

Staff Operations

Project Overview

Designed with Jetpack Compose, this system was built to handle high-traffic registration and attendance recording for Ceramica Market exhibitions. It bridges the gap between digital registration and physical attendance with speed and reliability.

Key Features

  • Integrated QR Tech: Staff can instantly verify tickets and record attendance via camera scanning.
  • Direct Communication: Admins can initiate calls to exhibition staff directly through the application's user directory.
  • Hybrid Registration: Supports on-site new user onboarding as well as database lookups by phone or email.

Core Impact

The app ensures a professional and organized experience for visitors while providing exhibition organizers with actionable real-time data on visitor numbers and staff performance.

Technologies
