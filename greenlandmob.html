GreenLand Management System

Enterprise Exhibition & Space Logistics Solution

Admin Workspace

Employee Workspace

The Vision

Developed specifically for Greenland CO, this application optimizes the management of exhibition areas. It eliminates double bookings and provides a centralized hub for reservation tracking.

Key Capabilities

  • Area Logic: Add, edit, and track status (Available, Pending, Sold).
  • Smart Security: Client data is strictly visible only to the assigned employee and Admin.
  • Real-time Sync: Built on Firebase for instantaneous updates across all devices.

Use Case

An employee searches for a space meeting a client’s needs, marks it as "pending," and the Admin receives an instant notification to review and confirm the reservation.
